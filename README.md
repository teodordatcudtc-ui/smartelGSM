# Smartel GSM - Site Web

Site web profesional pentru service GSM în București, Crângași.

## Tehnologii

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Instalare

1. Instalează dependențele:
```bash
npm install
```

2. Rulează serverul de dezvoltare:
```bash
npm run dev
```

3. Deschide [http://localhost:3000](http://localhost:3000) în browser.

## Build pentru producție

```bash
npm run build
npm start
```

## Structura proiectului

- `app/` - Pagini și layout-uri Next.js
- `components/` - Componente reutilizabile
- `public/` - Fișiere statice
- `app/globals.css` - Stiluri globale și Tailwind

## Pagini

- `/` - Pagina principală
- `/servicii` - Lista serviciilor
- `/despre-noi` - Despre Smartel GSM
- `/contact` - Formular de contact și informații
- `/politica-confidentialitate` - Politică de confidențialitate
- `/termeni-conditii` - Termeni și condiții
